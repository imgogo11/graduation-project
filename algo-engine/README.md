# Algo Engine

This directory hosts the C++ analysis engine and Python binding layer for the graduation project.

## Planned layout

- `include/algo_engine/`: public headers grouped by algorithm family
- `src/`: C++ implementation files
- `bindings/python/`: pybind11 binding layer
- `tests/`: unit and integration test entry points
- `benchmarks/`: performance comparison entry points

## Initial scope

- C++23 project skeleton
- `algo_engine_core` target placeholder
- `algo_engine_py` target placeholder
- test target hook for future algorithm validation
